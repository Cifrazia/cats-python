# CATS Python

## Cifrazia Action Transport System

### Requirements

+ Python `^3.9`
+ Tornado `^6.1`
+ Sentry SDK `^1.1.0`
+ uJSON `^4.0.2`
+ PyTZ `^2021.1`

**Extras**

+ `django`
  + Django `^2.2` (optional)
  + Django Rest Framework `^3.12.4` (optional)

### Installation

**Install via Poetry**

```shell
poetry add cats-python
```

With `Django` support:

```shell
poetry add cats-python[django]
```
